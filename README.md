# ☁️ CloudSim Lab - Simulação de Infraestrutura em Nuvem

> **Atividade Prática** | Kali Linux + NetBeans + CloudSim  
> _Documentação oficial do projeto_

---

## 🎯 Propósito

Demonstrar na prática como simular uma infraestrutura de nuvem usando o framework CloudSim, criando um Datacenter virtual com VMs e tarefas (Cloudlets), sem custos de hardware real.

---

## 🛠️ Tecnologias Utilizadas

| Ferramenta      | Versão  | Finalidade               |
| --------------- | ------- | ------------------------ |
| Kali Linux      | Rolling | Sistema operacional base |
| Java JDK        | 17      | Runtime e compilação     |
| Apache NetBeans | 15      | IDE de desenvolvimento   |
| CloudSim        | 7.0.1   | Framework de simulação   |
